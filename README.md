# 1.現代社会の問題点
日本において個人主義が現在まかり通っておりますがこれ自体が問題です。例えばここに三種のタケシがいたとします。

①ニートタケシ

②フリータータケシ

③任天堂法務部タケシ

Q.じゃあどのタケシがtier高いか？

A.任天堂法務部タケシです。

と多くは答える事でしょう。個人主義とは「どいつもタケシなんだから変わんないでしょ」と言える人の事です。そういませんね。

このように俺達日本人は、社会主義の視点で人を評価しています。この上で個人主義で評価される部分を磨こう磨こうと言っているのが現代です。やれ自己実現とか言ってね。おかしいでしょ。

この混沌とした時代を生き抜く為の人事管理フレームワーク？を考えてきました。

# 3.本題 設計【超有能と普通と無能の間に隔たりを作ろう】
一緒に仕事はするんですけど、ノルマの基準は共有しないよというのがここの肝です。(ノルマを共有すると足を引っ張る事に合理性ができちゃうゾ♡)

名前を付けるのは読者のあなたにおまかせ♡するとして、ここでは仮に

・S層→特段有能か、または異質な才能を持つ者　別のファイルではCOEって書いてます。

・A層→S層を除いた上位７割、一般大衆

・B層→S層を除いた下位３割、潜在的ルサンチマンか、A層にいたけどなんかあってやる気なくしちゃった人達

・A層とB層は業種にもよりますが月イチで変動します。

とします。

# 3-1.S層について
・指名制、又は推薦でしか入れない

・すべての仕事にアクセスできる

・すべての仕事の進捗状況を観測できる

・昇進や降格はしない。特異課とか名前つけるとかっこいいですね？

・役職も付かない

・緊急時対処とか、やべー現場のヘルプとかする役

・評価は赴いた現場の満足度で決まる

・給与体系は「A層の給与平均*0.7~2(現場の満足度により上下)」

S層は超有能！だったり異質な能力を持っている人がここに入ります。俺みたいなフレームワーク作る能力を持った人とかここに入れてほしいなーって、高望み？その辺は皆さんが決める事ですね。業種によるので。

そして、ただ一つS層だけの基準があるとするなら、それはS層の推薦、指名がある事でしかS層に入れないという事です。

あまりにも才能がある人たちは普通の人達を脅かしてしまいます。どれくらい脅かしてるかとかアンケで取ってS層に送り飛ばそう！とかやったらしょんぼりしちゃいそう。俺ならする

なので指名制です。S層に元からいる人は多分そう極まった人を見たなら顔でわかります。

それと、すべての仕事にアクセスできます。なんのヘルプだってできますし、何をやるのも基本自由です。

緊急時やこの現場そろそろヤバそうって所はS層にヘルプを出す事ができます。その時はそっち優先です。

それと、S層だけ評価基準が違います。助けに行った現場からの満足度で評価します。

なぜなら天才というのは得てして他人を傷つけてしまいがちだからです。

5人分働いて10人分士気を落としました！自分はたくさん仕事したし、事態は解決したけどヘルプに行った所が会社の事うっすら嫌いになりました！では天才が会社にいる意味がありません。

そういった事態を防ぐために現場の満足度のみを評価対象とします。

# 3-2.A層について
・上位7割(ここの割合は任意。俺は7割か8割をおすすめします)

・平社員から部長とかお偉いさん、社長まで満遍なく入る

・B層よりノルマも給料も高い

普段フツーに仕事してる皆さんがここに該当します。

評価はこれまで通り！仕事内容も出世街道もこれまでと同じ！

生活が変化するという事は何もないと思います。

しいて言えばこれまでより生活がちょっと楽になるんじゃないでしょうか？天才の見下しも、こじれた人からうまく距離を取る必要もなくなるので。

# 3-3.B層について
・下位3割

・やる気が無くなった人が落ちてくる

・頑張り次第でA層への上昇も可

・ノルマは最低限、給料もそこそこ

潜在的ルサンチマン層です。彼らをほっぽらかしにしておくと、みんなの士気を下げに来たりとかします。

失敗した後とか、長期的に攻撃的になったり燃え尽きちゃった人が羽休めする層にもなります。最悪一生出てこなくてもいいし、頑張って出てくることもあるでしょう。この要因として考えられるのは己との葛藤の勝利以外あり得ないので、会社はなんもできません。

同様に先天的にどうしようもなく能力のない人など、社内平均で劣る人が入ります。

ノルマが少なく、給料もA層ほどはなくてもある程度そこそこで安定して疲れず仕事できればいいという層です。

無限にgeminiと壁打ちしている間に分かったことですが、ここ経営のエラーをいち早く捉えてくれる場所ですね。ビッグモーなんちゃらとかいう会社では下に押し付けることが横行した為に、多分下から崩れていったのだと思います。

もし下3割の領域と上7割の領域が大幅に変動することがあった場合、それは会社の緊急事態です。いちはやくノルマを下げてね下げてね下げてね

下3割が最低限のノルマを達成できなくなった時もまあ緊急事態です。その時は多分ノルマを押し付けられたり全体的に無理させられてる証左です。

ちなみにまとめてノルマ未達がごっそり増えた場合は結託されてるかもしれません。その辺は気を付けてネ♡

Ｓ層がどこヘルプ行こっかなーする時に役立ちます。もちろん仕事してる人のモチベにもなります。

# 4-2.働くＡ層の人の嫁に、Ａ層が働けば働くほどポイントを与えてほしい！(あと旦那応援スタンプと嫁に感謝スタンプを相互に送れるようにしてください)

んで感謝して！できれば褒めて！を限りなく遠回しに、なぜなら相手の意思で言って欲しい！というフィルターを通すと、「専業主婦は忙しくてつらい」になります。
